# Visual Brief — career-school LP

> **主力ペルソナ：中村 早紀（29歳・法人営業・品川区一人暮らし）**
> 「営業向いてる気はする。でも、ずっとこの働き方は無理かも」

このブリーフは LP に必要な画像・動画素材を **生成AIですぐ作れるプロンプト形式** で記述したもの。各素材は Midjourney / Nano Banana / Sora / Runway / Kling 等にそのまま貼り込める形で整備。

---

## 1. デザイン方針（必ず守ること）

| 軸 | 採用 | NG |
|---|---|---|
| トーン | 暮らし美学系・ドキュメンタリー風・UGC風 | 広告然・スタジオ然・キラキラ過剰 |
| モデル | 等身大の20代後半〜30代前半日本人女性 | 海外人モデル・10代・40代以上 |
| 表情 | 静かな前向き・自然な微笑・ふと考える顔 | 過剰な笑顔・営業スマイル |
| 光 | 自然光（窓辺・午後）・暖色寄り | スタジオ蛍光灯・真昼の硬い光 |
| 背景 | 都内マンション・カフェ・コワーキング・移動中 | 真っ白背景・装飾過多 |
| 服装 | スーツ、リネンシャツ、ニット、デニム | 派手な制服感・着物 |
| 撮影スタイル | 35mm風・浅い被写界深度・少しドキュメンタリー | スマホ自撮り感・正面ガッツリ |

## 5. 撮影/生成の優先順位

予算が限られる場合、この順で揃える：

| 優先度 | 素材 | 理由 |
|---|---|---|
| 🥇 必須 | A-01 (FV) | 第一印象を決める。CVRに直結 |
| 🥈 高 | C-01〜C-04（4枚） | ストーリーセクションは LP の心臓部 |
| 🥈 高 | V-01（FV直下動画） | 動画ありなしで FV 滞在時間が大きく変わる |
| 🥉 中 | F-01〜F-04（受講生顔） | 信頼形成 |
| 🥉 中 | D-01〜D-03（ベネフィット帯） | 説得力 |
| ▽ 低 | E-01〜E-08（スキルカード） | 図解で代替可能 |
| ▽ 低 | A-02 / A-03（FV B/C） | A/Bテスト用なので後追いでOK |
| ▽ 低 | V-02 / V-03 / V-04 | 動画は1本でも十分機能する |

## 6. 一貫性チェックリスト（納品時に必ず確認）

- [ ] 全画像のホワイトバランス・カラーグレーディングが揃っている
- [ ] 主人公モデル（F-01）= ストーリー（C-01〜C-04）= FV（A-01）が **同一人物**
- [ ] 配色がブランドパレット（#2E2A28 / #C2543F / #D4B896 / #F5EDE0 / #FFE4A3）の範囲内
- [ ] 「直球の煽り」表情・ポーズが入っていない
- [ ] 男性が画面に入っていない（女性向けスクール特化のため）
- [ ] スマホ画面・PC画面に **実在ブランドのロゴ・UI** が映っていない
- [ ] 動画は全て無音・無限ループ可能・5-15秒
- [ ] WebP 出力 + PNG/JPG フォールバック用意
- [ ] Retina対応（@2x = 表示サイズの2倍ピクセル）

## 8. 法的留意

- 受講生として顔写真使用する場合は **必ずモデルリリース取得**
- AI生成画像でも「※イメージ画像」「※AI生成」表記を推奨（景品表示法上のリスク低減）
- 「お客様の声」は **実在する受講生の同意ある実体験のみ** 載せる
- 数字（合格率・年収・受講生数）には **n数・調査期間・出典** を必ず脚注で記載
